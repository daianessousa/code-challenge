### Starting the project

```bash

# Clone the repository 
$ git clone git@github.com:daianessousa/code-challenge.git

# Install the dependencies
$ yarn

# Run project
$ yarn start

#Run the storybook 
$ yarn storybook

# Will start on http://localhost:3000/ 

```
